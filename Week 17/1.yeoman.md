基本使用

- npm init
- npm install yeoman-generator  
- npm install -g yo
- cd /
- npm link 全局链接  可以链接具体某个包https://www.bbsmax.com/A/kmzL79KYdG/
- yo toochain

> cnpm 安装的镜像用不了，需要用npm安装才link上
> generators  写成genertor是不行的
> package.json name 需要使用generator开头
> package.json main 需要修改
